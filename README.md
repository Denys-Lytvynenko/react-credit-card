# Project info

Reusable card component with ability of customization card view.

# Run project

1. First of all you need to install dependencies from npm:

```
npm install
```

2. Run project via command:

```
npm start
```

# Supported payment systems

-   4149 - Visa
-   5168 - Mastercard
-   6222 - Unionpay
-   3782 - AMEX
-   60110 - Discover
